Basic Calcualtor

  1. Site Name: World Simple Calculator (WSC).

    The name given to this calculator reflects its purpose. It is a very simple calculator can be used online for a daily simple computations.

  2. Site Purpose: For World Simple Calculator.

    This project is mean to provide some fundamental basic computation that is found everywhere. Making it freely available online and by means of a web-based will make it easy to access.

  3. Scenarios:
    • Is the calculator truly freely avalaible ?
    • Does it sotre computation result ?
    • Does it have dark mode ?
  4. Color Schema:
    • Primary color: #10335c. This will be used on the header and footer.
    • Secondary color: #275172. This will be applied to the key and button of the calculator.
  5. Typography:
    • Heading Font: 'Merriweather', serif.. These will be applied to the headings.
    • Body Font: "Roboto", sans-serif .. These will be applied to the body of the site.
