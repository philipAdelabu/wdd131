Documentation

This is a brief documentation of the usage of the simple calculator. It is very simple to use.


  1. Site Name: World Simple Calculator (WSC).

    The name given to this calculator reflects its purpose. It is a very simple calculator that can be used online for a daily simple computation.

  2. Site Purpose: For World Simple Calculator.

    This project is mean to provide some fundamental basic computation that is found everywhere. Making it freely available online and by means of a web-based will make it easy to access.

  3. Screen Display:
    • There are two separate layers of the display.
    • Input display: This dispaly the key-pressed numerical and symbol value.
    • Output display: This is the output result display and error message display.
  4. Keypad
    • Numeric [0 - 9]: These are numbers available for computation.
    • Operators [*, -, /, =, +]: These are the computating operators for basic calculation.
  5. Control Keys
    • STR: Use to store the last computational result in the memory.
    • RCL: Use to retrieve the last stored result from the memory.
    • CLR: Clear the display output from both screen dispaly layers.
